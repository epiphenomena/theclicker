# Contributing

## File Organization

- **src/**: All source code
- **assets/**: Static assets (images, sounds)
- **tests/**: Test files

## Code Style

- Use ES6+ features
- Modular functions
- Clear variable names
- Comment complex logic

## Adding New Features

1. Create appropriate subdirectory in src/ or assets/
2. Write modular, testable code
3. Add tests in tests/
